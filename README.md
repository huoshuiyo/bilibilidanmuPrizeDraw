# BilibilidanmuPrizeDraw

#### 介绍
B站弹幕抽奖工具

#### 软件说明
本项目使用unity2019 4.32f1c1开发


#### 安装教程

自己打包

#### 基本功能

1.  弹幕显示 人气显示
2.  弹幕口令抽奖 可以限制粉丝牌等级
3.  单抽，十连抽，自定义抽取
4.  响指一声：去掉奖池里一半的人数（人数向下取整）
5.  CSGO单抽：模拟CSGO开箱的方式进行单抽
6.  舰长，礼物显示
7.  抽奖动画
8.  中奖记录，中奖者排除
